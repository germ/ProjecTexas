About

This is a showcase of various prisoners that have been sent to death row in Texas. For each convict their crimes will be presented alongside their final statements, in addition supplementary data from the Texas Department of Criminal Justice will be displayed. Feel free to select a name from the left to begin.

History

I came up with the idea for these pages after finding this database via Hacker News. In particular I found the dying words of convicts to be interesting, especially when contrasted against the violence of their crimes. I believed that this data should have been presented with more of a focus on this contrast and decided to showcase it here as such.

Technical Aspects

The data presented was scraped from the Texas Department of Criminal Justice website using scrapy and exported to JSON. After initial data collection the data was processed with a python script and "pickled" into raw text. This file was then loaded as a python dictionary using Google App Engine and parsed into the pages you see here with Django templates. If time would have permitted the hacked together pickling solution would have been replaced with a proper mysql database.
